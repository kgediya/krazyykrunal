SkyHi is a multi-surface spatial messaging project built around shared skies, lantern-based messages, and connected experiences across Spectacles, web, and companion surfaces.

What SkyHi Does

SkyHi lets users send a message into a shared Sky PIN and experience those messages as floating lanterns in immersive and companion environments.

  • Users can send lantern-style messages through the web companion.
  • The same shared sky can be opened in immersive experiences for viewing and interaction.
  • The system is designed to connect identity, messaging, and spatial presence across surfaces.

Snap Integration

SkyHi can use Snap platform features such as Camera Kit-enabled immersive experiences and optional Snapchat login for account-backed sign-in on the web companion.

  • Snapchat login is optional and used for authentication and approved identity fields only.
  • SkyHi does not request private Snapchat messages, contacts, or private shared content.
  • Users can still interact with SkyHi features without relying on sensitive personal data.

Who Can Use SkyHi

SkyHi is intended as a creative communication and spatial interaction experience. Users should avoid sharing sensitive personal information in messages.
